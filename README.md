# Interactive Resume Builder

## 📌 Description
A responsive web application that allows users to create, edit, and preview resumes in real time with a clean and user-friendly interface.

## 🚀 Features
- Live resume editing with instant preview
- Dynamic content rendering
- Form validation for accurate input
- Download/print-ready resume generation
- Structured and easy-to-use form interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Use
1. Enter your details in the form
2. Preview updates instantly
3. Download or print your resume
